# Grace Community Church Website

A welcoming, modern website built for Grace Community Church in Thohoyandou, Limpopo province. It is designed to help visitors find service times, watch sermons, and get involved.

##  Features  
- **Events section**: Upcoming services, outreaches, and youth programs
- **Donation**: Easy online giving integration
- **Contact form**: Messages go straight to church email + Google Maps
- **Light/Dark mode**: Toggle for accessibility

## 🛠️ Built With
HTML • CSS • JavaScript • No frameworks needed

##  File Structure
index.html, about.html,contact.html,enquiries.html ,service.html,style.css, script.js

## Pages
This website has different pages that helps the visitor to navigate through them easily like the service page where they will find different services offerd by the church.
